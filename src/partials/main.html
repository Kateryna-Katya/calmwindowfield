About the Game

Your own little creative playground

Calm Window: Dreamline Architect lets players design miniature dream environments inside framed “windows.” Each one evolves over time based on simple choices and gentle interactions.
It's not about winning or completing objectives - it's more about creating a vibe and expressing yourself.

How to Play

Create, adjust, observe

  • Open a Window

    Start with an empty frame waiting to be shaped.

  • Choose Elements

    Add weather, objects, colors, ambient details.

  • Let It Evolve

    Watch how your window changes over time.

  • Refine the Mood

    Adjust elements to guide the atmosphere.

  • Expand Collection

    Create multiple windows with different feelings.
